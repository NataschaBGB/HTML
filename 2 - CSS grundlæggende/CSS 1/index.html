Om CSS

Hvad er en selector med egne ord?

En selector er den man vælger hvad der skal ændres på. Altså *, #main, .question og .answer osv i denne opgave.

Hvad er en property med egne ord?

En property er den man definerer HVAD der skal ændres på din selector. F.eks. color, font-size, border og mange flere.

Hvad er en value med egne ord?

En value er den værdi man tillægger sin property. Color skal have en farve, font-size skal have et tal (15px)...
