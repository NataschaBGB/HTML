HTML 1

Hvad står HTML for?
HyperText Markup Language.

Hvem opfandt HTML?
HTML-sproget blev opfundet af fysikeren Tim Berners-Lee.

Hvilken version af HTML er den nyeste (og den vi skal bruge?)
HTML5

Hvad indeholder < body >?>
< body > indeholder alt det, som brugeren kan se på f.eks. en hjemmeside.

Hvad kan < head > blandt andet indeholde og hvad SKAL den indeholde?
< head > kan indeholde style og link.
Den SKAL indeholde title og metadata.

Hvorfor skal man helst kalde sin fil index.html?
index.html er et default navn. Hvis man vil ind på en hjemmeside,
og man ikke definerer hvilket navn den har, vælger serveren automatisk finde og vise defaulten.
Jeg søgte på 'why call the file index in html'.
